# System Project Types Reference

## Overview

CoHive ships with 20+ pre-configured project types, each with a professionally crafted AI prompt optimized for specific business use cases. These are automatically available to all users and cannot be modified.

**Source**: `/data/systemProjectTypes.ts`

## Usage

When running assessments in CoHive:
1. Select a project type from the dropdown
2. If it's a system project type, the corresponding AI prompt is automatically applied
3. The AI uses this prompt to guide its analysis and recommendations
4. No additional configuration needed - it's seamless!

---

## Adding Custom Project Types

Data Scientists can create additional project types with custom prompts:
- Navigate to Knowledge Base → Synthesis → "New Project Type"
- Custom types must have unique names (cannot conflict with system types)
- See `/docs/PROJECT_TYPE_PROMPTS.md` for details
